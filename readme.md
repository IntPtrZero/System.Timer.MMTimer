# System.Timer.MmTimer

## A High - Accurancy timer for .NET

### 特点(Features)
* 高精度(最低1~2ms)
* 支持Winform设计器
* 封装同步上下文，解决跨UI线程操作问题
* 封装常规属性与方法，与普通Timer保持一致

### 使用方法(How to)
    引用程序集，即可在工具栏找到MMTimer组件  
<br>

*参见[MSDN - Multimedia Timer](https://docs.microsoft.com/en-us/previous-versions//ms712704%28v%3dvs.85%29)*
